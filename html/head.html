HTML Head

1.The HTML head tag is used for indicating the head section of the HTML document
2.The head element is a container for metadata (data about data).
3.It is placed between the html tag and the body tag.
4.HTML metadata is data about the HTML document.
5.Metadata is not displayed.
6.Metadata typically define the document title, character set, styles, links, scripts, and other meta information.
